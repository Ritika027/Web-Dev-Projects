# 📚 Unacademy Website Clone (HTML & CSS)

This is a **frontend clone** of the **Unacademy** homepage, built using only **HTML and CSS**.  
It was created to practice and improve my web design skills by recreating the layout and look of a real educational platform.

---

## 🌟 Features

- Top navigation bar with logo and menu items
- Hero section with banners and highlights
- Sections for categories, educators, and courses
- Footer area with multiple useful links
- Clean and structured layout using Flexbox and Grid

---

## 🛠️ Technologies Used

- **HTML** – for structuring the webpage  
- **CSS** – for styling and layout design

---

## 📘 What I Learned

This project helped me understand:

- How to structure large web pages into sections
- How to design complex layouts with Flexbox and CSS Grid
- Styling buttons, cards, and typography
- Matching color schemes and spacing from real websites
- Improving my confidence in using pure HTML and CSS
