# 🔢 Guess The Number Game (HTML, CSS & JavaScript)

This is a fun and interactive **Guess the Number** game built using **HTML**, **CSS**, and **JavaScript**.

The player has to guess a randomly generated number within a specific range. The game provides feedback like "Too High", "Too Low", or "Correct!" and keeps track of the score.

---

## 🎮 Features

- Random number generation on each game start
- Input field for guessing the number
- Real-time feedback on guess attempts
- Score tracking system
- Option to restart the game
- Styled UI with basic animations and effects

---

## 🛠️ Technologies Used

- **HTML** – for structure  
- **CSS** – for styling  
- **JavaScript** – for logic and interactivity
