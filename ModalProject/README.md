# 💬 Modal Popup Project (HTML, CSS & JavaScript)

This project is a simple and elegant **Modal Popup** built using **HTML**, **CSS**, and **JavaScript**.  
It demonstrates how to display and close a modal box when triggered by a button, commonly used for alerts, forms, or messages on websites.

---

## 🌟 Features

- Click a button to open the modal
- Modal appears centered with a dark overlay
- Close the modal using a close button or by clicking outside
- Smooth transitions and animations
- Clean and responsive design

---

## 🛠️ Technologies Used

- **HTML** – for structure  
- **CSS** – for layout, styling, and transitions  
- **JavaScript** – for interactivity and event handling
